# 商業模式

## 商業閉環

```
實體桌遊體驗（NT$1,200–1,500）
    ↓
掃碼輸入生辰八字
    ↓
免費五行基礎報告（AI 生成）
    ↓
付費完整 AI 命盤報告（NT$500–1,200）
    ↓
一對一諮詢（NT$2,000–5,000）
```

## 三階段規劃

### 第一階段：自營工作坊
- 自己帶工作坊，單場 **NT$800–1,200 / 場**
- 目標：驗證遊戲節奏、收集玩家真實回饋、累積第一批命盤報告使用數據

### 第二階段：教練認證
- 教練認證課程 **NT$10,000–15,000 / 人**
- 授權年費 **NT$3,000–5,000 / 年**
- 教練可用自己的社群辦工作坊，回饋給平台玩家命盤資料

### 第三階段：訂閱制社群
- 月費 **NT$199–399 / 月**
- 提供進階命盤、社群互動、定期工作坊回訪

## 商業優勢

- **桌遊作為獲客**：低門檻的體驗式行銷，玩家自發分享
- **命盤報告作為後端**：高毛利，AI 生成邊際成本極低
- **教練模式可規模化**：不必每場都自己帶
- **單人模式增加黏著度**：同一套牌可重複使用

## 已知風險

- 命理報告品質高度依賴 AI Prompt 工程，需要系統化測試
- 教練認證課程內容開發成本高
- 第一場工作坊前所有實體物件必須到位
- 支付整合（綠界／Stripe）與後台用戶管理需額外開發
