# 主持人工作表

## 開場逐字稿

> 你以為你了解自己。但今天，你說出口的第一句真話，可能是你第一次真正聽見自己。準備好了嗎。
>
> 等一下你會抽到一張季節卡。這張卡說的是你今天帶來的能量狀態，不是你永遠的樣子。同一個人，不同的天抽到不同的牌，都是真的。
>
> 我唯一的要求是——試著給第二個版本。不用很深，但要是真的。

## 轉折點逐字稿

> 這是你今天帶來的五行能量。看著這張卡上的描述，有沒有哪一句話說的是今天的你？
>
> 這是你今天直覺帶來的五行。等一下遊戲結束之後，你可以掃碼看看你真實命盤的五行是什麼——跟今天一樣嗎？

## 結尾逐字稿

> 今天抽到的牌，不是巧合。你的命盤早就寫好了——只是你還沒看過。掃這個碼，第一次真正看見你自己。

## 單人模式抽牌引導

> 你手上這張牌是隨機的。但它說的，是你一直知道卻沒有說出口的事。翻開它。

## Checklist

### 開場
- [ ] 根據人數從 12 張季節卡隨機抽出對應張數，其餘收起
- [ ] 洗牌：秘密任務卡、十神卡分開洗
- [ ] 每人發 3 顆共鳴籌碼
- [ ] 每人發季節卡（公開）
- [ ] 每人發秘密任務卡（秘密）
- [ ] 說開場逐字稿

### 第一階段
- [ ] 翻暖場卡 3 張（一亮）
- [ ] 翻深入卡 2 張（二亮）
- [ ] 翻點火卡 1 張（三亮）

### 轉折點
- [ ] 根據對照表把五行卡放在每個人面前
- [ ] 說轉折點逐字稿
- [ ] 說掃碼預告

### 第二階段
- [ ] 看桌上五行組合，對照生剋表選卡
- [ ] 說邀請語再點名
- [ ] 按人數決定用幾張

### 第三階段
- [ ] 每人抽十神卡
- [ ] 給 2 分鐘安靜閱讀
- [ ] 每人說像不像自己

### 結尾
- [ ] 每人說「今天我發現自己＿＿＿」（每人 45 秒）
- [ ] 每人揭曉秘密任務（每人 30 秒）
- [ ] 說結尾逐字稿
- [ ] 發覺察卡
- [ ] 引導掃碼

## 緊急接話句

| 玩家狀態 | 主持人說 |
|---------|---------|
| 「我不知道」 | 「你第一個念頭是什麼？不用對，說出來就好。」 |
| 沉默超過 10 秒 | 「不用想太深，你現在腦子裡有什麼？」 |
| 用笑話帶過 | 「這個答案是給別人的，那個給自己的版本是什麼？」 |
| 說很表面 | 「如果再深一點，你會怎麼說？」 |
| 拒絕回答 | 「沒關係，你可以只是聽。這一輪你負責觀察。」 |
